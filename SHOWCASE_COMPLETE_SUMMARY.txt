🐛 TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────

Common Issues:

1. "ModuleNotFoundError: No module named 'showcase_data'"
   → Make sure showcase_data.py is in the same folder

2. "No module named 'pandas'"
   → Run: pip install pandas matplotlib seaborn tabulate

3. "Jupyter command not found"
   → Run: pip install jupyter

4. Charts don't display in notebook
   → Run: pip install --upgrade matplotlib seaborn

5. Thai text shows as boxes
   → Normal for fallback fonts; functionality works fine
