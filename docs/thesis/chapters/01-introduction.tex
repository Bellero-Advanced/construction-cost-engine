\chapter{บทนำ}

\section{ที่มาและความสำคัญ}
\label{sec:background}

งานก่อสร้างภาครัฐมีบทบาทสำคัญต่อการพัฒนาโครงสร้างพื้นฐานและเศรษฐกิจของประเทศ
\autocite{sirieawphikul2024}.
การจัดทำงบประมาณโครงการก่อสร้างจำเป็นต้องอาศัยการประมาณราคาที่มีความถูกต้อง
และสอดคล้องกับสภาพต้นทุนก่อสร้างจริงและสามารถตรวจสอบให้เป็นไปตามมาตรฐาน
การคำนวณต้นทุน \autocite{crone1992}
โดยเฉพาะการกำหนดราคาต่อหน่วยของวัสดุก่อสร้าง ซึ่งเป็นองค์ประกอบหลักในการคำนวณ
ราคากลางและงบประมาณของโครงการ \autocite{ashworth2013}.

ในปัจจุบัน หน่วยงานภาครัฐนิยมใช้ราคาวัสดุก่อสร้างที่ประกาศโดยกระทรวงพาณิชย์
เป็นข้อมูลอ้างอิงหลัก เนื่องจากเป็นแหล่งข้อมูลที่มีความน่าเชื่อถือและได้รับ
การยอมรับอย่างเป็นทางการ อย่างไรก็ตาม ข้อมูลราคาดังกล่าวยังมีข้อจำกัดด้าน
ความครอบคลุมทั้งในด้านพื้นที่ ช่วงเวลา และประเภทของวัสดุก่อสร้าง ส่งผลให้ใน
บางโครงการไม่สามารถนำมาใช้คำนวณราคาต่อหน่วยได้โดยตรง \autocite{leung2005}.
ผู้ประมาณราคาจึงจำเป็นต้องสืบค้นข้อมูลตามมาตรฐานการคำนวณราคากลาง เช่น
การหาวัสดุก่อสร้างแหล่งอื่น ซึ่งกระบวนการดังกล่าวใช้เวลามีขั้นตอนซ้ำซ้อน
และอาจขาดความเป็นมาตรฐานเดียวกัน \autocite{benchanakatkun2014}.

ขณะเดียวกัน ร้านค้าปลีกวัสดุก่อสร้างขนาดใหญ่ได้มีการประกาศราคาวัสดุก่อสร้าง
อย่างเป็นระบบและปรับปรุงข้อมูลอย่างสม่ำเสมอ ซึ่งสามารถสะท้อนสภาพตลาดจริง
ในแต่ละพื้นที่ได้ หากสามารถนำข้อมูลจากแหล่งต่าง ๆ มารวบรวมและประมวลผลร่วมกัน
อย่างเป็นระบบ จะช่วยเพิ่มความครบถ้วนและความแม่นยำของข้อมูลราคาวัสดุก่อสร้าง.

\section{ปัญหางานวิจัย}
\label{sec:problem}

\begin{enumerate}
  \item ราคาวัสดุก่อสร้างจากแหล่งข้อมูลภาครัฐและภาคเอกชนกระจัดกระจาย
        ไม่มีระบบรวบรวมและเปรียบเทียบที่เป็นมาตรฐานเดียวกัน
  \item รูปแบบข้อมูลราคาในแต่ละแหล่งแตกต่างกัน — ทั้งหน่วยจำหน่าย
        ชื่อสินค้า และโครงสร้างเอกสาร — ทำให้การจับคู่และเปรียบเทียบทำได้ยาก
  \item กระบวนการสืบราคาในปัจจุบันยังเป็นแบบกึ่งอัตโนมัติหรือทำด้วยมือ
        ใช้เวลามาก และมีโอกาสเกิดความคลาดเคลื่อน
\end{enumerate}

\section{วัตถุประสงค์}
\label{sec:objectives}

การวิจัยเรื่องการพัฒนาระบบอัตโนมัติสำหรับการคำนวณต้นทุนต่อหน่วยของ
วัสดุก่อสร้างจากข้อมูลของกระทรวงพาณิชย์และสถานประกอบการค้าปลีกสมัยใหม่
ทั่วประเทศ มีวัตถุประสงค์ดังนี้

\begin{enumerate}
  \item เพื่อพัฒนาระบบในการดึงข้อมูลราคาวัสดุก่อสร้างจากกระทรวงพาณิชย์
        และเว็บไซต์ของ Modern Trade แบบอัตโนมัติ
  \item เพื่อพัฒนาระบบการคำนวณต้นทุนวัสดุต่อหน่วยตามหลักเกณฑ์การคำนวณ
        ราคากลางของภาครัฐ โดยเชื่อมโยงและเปรียบเทียบข้อมูลราคาวัสดุก่อสร้าง
        จากกระทรวงพาณิชย์และตลาด Modern Trade เพื่อนำมาวิเคราะห์ความแตกต่าง
        และปรับใช้กับรายการวัสดุตัวแทนใน BOQ
  \item เพื่อวิเคราะห์แนวโน้มและความสอดคล้องของต้นทุนวัสดุต่อหน่วย
        ในแต่ละประเภทตามช่วงเวลาและพื้นที่
\end{enumerate}

\section{ขอบเขตของการศึกษา}
\label{sec:scope}

\subsection{ศึกษาจากแหล่งข้อมูล}

\begin{itemize}
  \item ราคาจากกระทรวงพาณิชย์ ดัชนีราคาวัสดุก่อสร้าง (Construction Materials
        Index: CMI) รายจังหวัด/รายหมวดวัสดุ ข้อมูลรายเดือนย้อนหลังอย่างน้อย
        9 เดือน เพื่อวิเคราะห์แนวโน้มและความผันผวน
  \item ราคาจำหน่ายจริงจากเว็บไซต์ผู้ค้าปลีกสมัยใหม่ (Modern Trade) เช่น
        HomePro, MegaHome, Global House, Thai Watsadu, DoHome, Boonthavorn
  \item ครอบคลุมสินค้าหลัก 5--10 ประเภท ที่ใช้บ่อยและมีมาตรฐานชัดเจน
        (เช่น ปูนซีเมนต์, เหล็กเส้น, อิฐ, ทราย, กระเบื้อง, สีทาผนัง)
\end{itemize}

\subsection{ด้านวิธีการศึกษาและการวิเคราะห์ข้อมูล}

การศึกษานี้ใช้เทคนิคเว็บสแครปปิง (Web Scraping) ในการดึงข้อมูลราคาจาก
เว็บไซต์ และดำเนินการจัดการข้อมูล ได้แก่ การทำความสะอาดข้อมูล การปรับ
รูปแบบหน่วยวัด และการจับคู่รายการวัสดุก่อสร้างที่มีลักษณะใกล้เคียงกัน
การวิเคราะห์ข้อมูลเป็นการวิเคราะห์เชิงเปรียบเทียบและเชิงสถิติพื้นฐานเพื่อ
ศึกษาความแตกต่างและความสอดคล้องของราคาวัสดุก่อสร้างจากแหล่งข้อมูลต่าง ๆ.
