\section{การทวนสอบความถูกต้องของข้อมูลราคา (Data Validation)}
\label{sec:validation}

ก่อนนำข้อมูลราคาไปใช้ในการคำนวณต้นทุนต่อหน่วย จำเป็นต้องประเมินความสอดคล้อง
ของราคาที่ระบบรวบรวมได้จากแหล่งต่าง ๆ เพื่อให้มั่นใจว่าข้อมูลที่จัดเก็บ
ในระบบสะท้อนราคาตลาดจริงอย่างสมเหตุสมผล.

\subsection{วิธีการทวนสอบ}

ในช่วงทดลองระบบ (ณ 24 พ.ค.\ 2569) ระบบมีข้อมูลราคาที่ดึงสำเร็จจาก
3 แหล่งหลัก ได้แก่ CGD, DIT และ HomePro (บางรายการ). การทวนสอบ
ในงานวิจัยนี้ใช้วิธีเปรียบเทียบราคาข้ามแหล่งสำหรับวัสดุชนิดเดียวกัน
(cross-source consistency check) โดยอ้างอิง CGD เป็น baseline
เนื่องจาก CGD เป็นราคามาตรฐานที่ภาครัฐกำหนดและตรวจสอบได้จากเอกสาร
ราชการ \cite{cgd2569}.

\subsection{การตีความ}

\begin{itemize}[leftmargin=2em]
  \item ราคา DIT สูงกว่า CGD เฉลี่ย $+$5.8\% (median $+$4.6\%) ซึ่งสอดคล้อง
        กับลักษณะของ DIT ที่สำรวจราคาตลาดรายวัน ขณะที่ CGD กำหนดราคา
        มาตรฐานสำหรับงานราชการซึ่งมักต่ำกว่าราคาตลาดเล็กน้อย.
  \item ความแตกต่างสูงสุดพบในปูนซีเมนต์และกระเบื้อง ($+$11.4\%) ซึ่งเป็น
        วัสดุที่ราคาตลาดผันผวนตามอุปสงค์-อุปทานมากกว่าเหล็กเส้น.
  \item ข้อมูลจาก HomePro (4 รายการ) ยังไม่เพียงพอสำหรับการวิเคราะห์
        เชิงสถิติ เนื่องจากบางรายการมีความแตกต่างของหน่วยบรรจุภัณฑ์
        (เช่น ทรายขาย per ถุง 50 กก.\ ที่ HomePro vs per ลบ.ม.\ ที่ CGD).
  \item การทวนสอบเชิงลึกกับแหล่งค้าปลีกสมัยใหม่ครบทั้ง 8 แหล่ง
        เป็นงานวิจัยต่อยอดที่ต้องดำเนินการเมื่อระบบ AI agent research
        สามารถรวบรวมข้อมูลได้ครบถ้วนมากขึ้น.
\end{itemize}

\subsection{ข้อสังเกตเชิงวิศวกรรม}

ในมุมมองของวิศวกรประมาณราคา ผลการทวนสอบนี้แสดงว่า:
\begin{enumerate}[leftmargin=2em]
  \item ราคา CGD เหมาะสำหรับใช้เป็น \emph{lower bound} ของต้นทุนวัสดุ
        ในงานประมาณราคาเบื้องต้น.
  \item ราคา DIT ซึ่งสูงกว่า CGD ประมาณ 5--11\% สะท้อนราคาตลาดจริง
        ได้ดีกว่าสำหรับงานประมาณราคาขั้นสุดท้าย.
  \item ผู้ใช้ควรตรวจสอบหน่วยบรรจุภัณฑ์ก่อนเปรียบเทียบราคาข้ามแหล่ง
        โดยเฉพาะวัสดุที่ขายทั้งแบบ bulk และ retail.
\end{enumerate}
