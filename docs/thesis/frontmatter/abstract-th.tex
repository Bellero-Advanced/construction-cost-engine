\chapter*{บทคัดย่อ}
\addcontentsline{toc}{chapter}{บทคัดย่อ}

ปัจจุบันการเข้าถึงข้อมูลเว็บไซต์ทางอินเทอร์เน็ตได้รับความนิยมและสะดวกสบายมากขึ้น
จากการพัฒนาของเทคโนโลยีสมัยใหม่ โดยเฉพาะเทคนิค Web Scraping
ซึ่งสามารถนำมาใช้แก้ไขปัญหาในงานวิจัยและการประมาณราคาได้อย่างมีประสิทธิภาพ
โครงงานวิจัยนี้พัฒนาระบบอัตโนมัติเพื่อดึงข้อมูลราคาวัสดุก่อสร้างที่ประกาศ
บนเว็บไซต์ของกระทรวงพาณิชย์ตามจังหวัดที่มีโครงการก่อสร้าง
และจากร้านค้าปลีกสมัยใหม่ (Modern Trade) เช่น HomePro, MegaHome, บุญถาวร, ไทวัสดุ ฯลฯ
แล้วนำมาใช้ในการคำนวณต้นทุนต่อหน่วยของวัสดุก่อสร้าง
ระบบรองรับการจัดการข้อมูล ได้แก่ การทำความสะอาดข้อมูล การปรับรูปแบบหน่วยวัด
และการจับคู่รายการวัสดุที่มีลักษณะใกล้เคียงกัน

การวิเคราะห์ข้อมูลมุ่งเน้นการเปรียบเทียบราคาต้นทุนต่อหน่วยของวัสดุก่อสร้าง
ในเชิงพื้นที่และช่วงเวลา รวมถึงการประเมินความสอดคล้องของราคาที่ประกาศจาก
ภาครัฐกับราคาตลาดจริงจาก Modern Trade ผลการศึกษาช่วยให้เห็นแนวโน้ม
และความแตกต่างของราคาวัสดุจากแหล่งข้อมูลต่าง ๆ อย่างชัดเจน
ซึ่งสามารถนำไปใช้สนับสนุนการประมาณราคางานก่อสร้าง การวางแผนงบประมาณ
และการตัดสินใจของหน่วยงานภาครัฐได้อย่างมีประสิทธิภาพ
