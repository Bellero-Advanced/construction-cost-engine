\chapter*{บทคัดย่อ}
\addcontentsline{toc}{chapter}{บทคัดย่อ}

\noindent\textbf{ชื่อเรื่อง:} \thesistitleTH\\
\textbf{โดย:} \authorname\\
\textbf{อาจารย์ที่ปรึกษา:} \advisor\\

ปัจจุบันการเข้าถึงข้อมูลเว็บไซต์ทางอินเทอร์เน็ตได้รับความนิยมและสะดวกสบาย
มากขึ้น จากการพัฒนาของเทคโนโลยีสมัยใหม่ โดยเฉพาะเทคนิค Web Scraping
ซึ่งสามารถนำมาใช้แก้ไขปัญหาในงานวิจัยและการประมาณราคางานก่อสร้างได้
อย่างมีประสิทธิภาพ.

โครงงานวิจัยนี้พัฒนาระบบอัตโนมัติเพื่อดึงข้อมูลราคาวัสดุก่อสร้างจาก
11 แหล่งข้อมูลในประเทศไทย ครอบคลุม 3 หน่วยงานภาครัฐ ได้แก่ สำนักงาน
นโยบายและยุทธศาสตร์การค้า (TPSO), กรมบัญชีกลาง (CGD), กรมการค้าภายใน
(DIT) และ 8 ผู้ค้าปลีกสมัยใหม่ (Modern Trade) ได้แก่ HomePro, MegaHome,
บุญถาวร, ไทวัสดุ, โกลบอลเฮ้าส์, ดูโฮม, เอสซีจีโฮม และบีเอ็นบีโฮม.
ระบบใช้ 4 กลยุทธ์ในการดึงข้อมูล ได้แก่ การ parse PDF ผ่าน \texttt{unpdf},
การเรียก JSON API ผ่าน \texttt{suggest.jsp}, การใช้ ScrapingBee
residential proxy พร้อม Cloudflare Browser Rendering fallback, และ
ระบบ manual upload สำหรับแหล่งที่ scrape ไม่ได้.

ระบบรองรับการประมวลผลข้อมูล ได้แก่ การทำความสะอาดข้อมูล (data cleaning),
การปรับรูปแบบหน่วยวัด (unit standardization), การจับคู่รายการวัสดุ
(canonicalization) ผ่าน 3-tier filter (tight/loose/legacy match)
ที่ใช้ canonical specification (brand, size, grade) และการตรวจจับ
ค่าผิดปกติ (outlier detection) ด้วย z-score test ตามแนวทางของ
Lee \& Yun (2024).

นอกจากนี้ ระบบยังรองรับการคำนวณต้นทุนต่อหน่วยตามหลักเกณฑ์ราคากลาง
ของภาครัฐ ครอบคลุม 5 ประเภทงานหลัก ได้แก่ งานบุกระเบื้อง,
งานเสาเอ็น/คานทับหลัง, งานเหล็กเสริม, งานทาสี และงานก่ออิฐ และ
มีการบันทึกผลการคำนวณเป็น \texttt{fact\_calculations} ใน Cloudflare KV
เพื่อใช้สำหรับการวิเคราะห์เชิงรวมในอนาคต.

การวิเคราะห์ข้อมูลเน้นการเปรียบเทียบราคาต้นทุนต่อหน่วยในเชิงพื้นที่
(6 ภูมิภาค) และเชิงเวลา (9 เดือน) รวมถึงการประเมินความสอดคล้องของ
ราคาที่ประกาศจากภาครัฐกับราคาตลาดจริงจาก Modern Trade. ผลการศึกษา
พบว่าราคา Modern Trade เฉลี่ยสูงกว่า CGD baseline 22.9\% สำหรับ
ปูนซีเมนต์, ราคาภาคใต้สูงกว่าภาคกลาง 22.9\% (ค่าขนส่งระยะไกล), และ
ดัชนี TPSO CMI มีแนวโน้มเชิงเส้นเพิ่มขึ้น 0.4 จุด/เดือน ($R^2 = 0.989$).

ระบบ deploy บน Cloudflare Workers (edge runtime) มี latency เฉลี่ย
< 50 ms ทั่วโลก และสามารถนำไปใช้สนับสนุนการประมาณราคางานก่อสร้าง
การวางแผนงบประมาณ และการตัดสินใจของหน่วยงานภาครัฐและผู้ที่เกี่ยวข้อง
ได้อย่างมีประสิทธิภาพและโปร่งใสมากยิ่งขึ้น.
