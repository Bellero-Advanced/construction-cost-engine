# src/lib/units/

## หน้าที่
แปลงหน่วยราคาวัสดุก่อสร้างระหว่างหน่วยต่าง ๆ เพื่อให้เปรียบเทียบข้ามแหล่งได้

## ไฟล์

### `converter.ts` — Unit Converter

**ปัญหาที่แก้:** แต่ละแหล่งขายวัสดุในหน่วยต่างกัน เช่น:
- CGD: ทราย = 480 บาท/ลบ.ม.
- HomePro: ทราย = 82 บาท/ถุง 50 กก.
- MegaHome: ทราย = 2,690 บาท/ถุง (ขนาดต่างกัน)

**หน่วยที่รองรับ 12 หน่วย:**

| กลุ่ม | หน่วย |
|-------|-------|
| มวล | ตัน, กก., กรัม, ถุง 50 กก., ถุง 20 กก., ถุง 1 กก. |
| ปริมาตร | ลบ.ม., ลิตร, แกลลอน (3.785L) |
| ความยาว | เมตร, ซม. |
| พิเศษ | เส้น 12 ม. (rebar), เส้น 10 ม. |

**สูตรแปลง rebar (per bar → per ton):**
```
weight_per_bar = π × (diameter/2)² × 7850 kg/m³ × length
price_per_ton = price_per_bar × (1000 / weight_per_bar)
```

ตัวอย่าง: HomePro DB12 = 359 บาท/เส้น 10 ม.
- น้ำหนัก = π × 0.006² × 7850 × 10 = 8.88 กก./เส้น
- ราคา/ตัน = 359 × (1000/8.88) = 40,436 บาท/ตัน
